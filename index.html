Reset.css

Reset.css добавляється у Styles.css таким чином:

a {

    text-decoration: none;

}

ul {

    list-style: none;

    margin: 0;

    padding: 0;

}

img {

    display: block;

}

h1, h2, h3, h4, h5, h5, h6, p {

    margin: 0;

    padding: 0;

    font-weight: 400;

}
